# FartLabs Webring

A webring for FartLabs members and friends. Hand-crafted sites welcome.

## Join the Webring

1. Fork this repo
2. Edit `webring.json` and add yourself to the `sites` list:
   ```json
   { "name": "your-name", "link": "your-site.com" }
   ```
3. Submit a pull request

## Rules

- Hand-crafted wikis, portfolios, and personal sites only
- No JS required (but fine if your site uses it)
- Be excellent to each other

## Member Sites

Visit [webring.fartlabs.org](https://webring.fartlabs.org) to browse the ring.
